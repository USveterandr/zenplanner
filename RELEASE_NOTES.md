# Zen Planner - Release Notes

## Version 1.0.0 (Initial Release)

**Release Date:** March 1, 2026
**Version Code:** 1
**Package:** com.zenplanner.app

### What's New

Zen Planner is an AI-powered productivity app designed to help you beat procrastination and take control of your day.

### Features

- **Smart Task Management** - Create, organize, and prioritize tasks with an intuitive interface
- **AI Advisor** - Get intelligent productivity recommendations and task suggestions
- **Calendar View** - Visualize your schedule and plan ahead
- **Goal Tracking** - Set and monitor progress on your personal and professional goals
- **Habit Builder** - Build positive habits with streak tracking and reminders
- **Analytics Dashboard** - Gain insights into your productivity patterns
- **Offline Support** - Access your tasks and plans even without an internet connection
- **Notifications** - Stay on track with timely reminders

### Technical Details

- Built as a Trusted Web Activity (TWA) wrapping https://zenplanner.vercel.app
- Minimum Android version: 5.0 (Lollipop, API 21)
- Target Android version: API 35
- App size: ~1.8 MB
